# OptiMPa — Concrete Compressive Strength Predictor

A full-stack machine learning web application that predicts concrete compressive strength (MPa) from mix design parameters.

**Stack:** Python · Scikit-Learn · FastAPI · Next.js · TypeScript · Tailwind CSS

---

## What It Does

You enter 8 concrete mix design parameters (cement content, water, aggregates, curing age, etc.) using interactive sliders. A trained **Random Forest Regressor** model predicts the compressive strength in MPa and automatically classifies it into an **EN 206 concrete grade** (e.g. C25/30, C40/50).

### Features
- 8 animated slider inputs with engineering context notes
- Instant MPa prediction via a FastAPI REST endpoint
- EN 206 grade classification with colour-coded result display
- Dark-mode-only minimalist UI
- Interactive API docs at `/docs` (Swagger UI)

## Running Locally

### Prerequisites

- Python 3.10+
- Node.js 18+
- npm

---

### Step 1 — Train the Model

```bash
cd ml_pipeline
pip install -r requirements.txt
python train.py
```

This generates `api/model.pkl` (~4.5 MB). Takes about 10–15 seconds.

---

### Step 2 — Start the API

Open a terminal:

```bash
cd api
pip install -r requirements.txt
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

API will be available at:
- **Prediction endpoint:** `http://localhost:8000/predict`
- **Interactive docs:** `http://localhost:8000/docs`
- **Health check:** `http://localhost:8000/health`

---

### Step 3 — Start the Frontend

Open a second terminal:

```bash
cd frontend
npm install
npm run dev
```

Open **`http://localhost:3000`** in your browser.

## API Usage

```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "cement": 350,
    "slag": 0,
    "fly_ash": 0,
    "water": 175,
    "superplasticizer": 6,
    "coarse_agg": 1040,
    "fine_agg": 780,
    "age": 28
  }'
```

**Response:**
```json
{
  "strength_mpa": 46.13,
  "strength_grade": "C40/50",
  "input_summary": { ... }
}
```

## Model Details

| Property | Value |
|---|---|
| Algorithm | Random Forest Regressor |
| Dataset | UCI Concrete Compressive Strength (I-Cheng Yeh, 1998) |
| Training samples | 440 |
| Test R² | ~0.84 |
| Test RMSE | ~7.5 MPa |
| CV R² (5-fold) | ~0.86 ± 0.03 |

## Dataset

UCI ML Repository — Concrete Compressive Strength  
https://archive.ics.uci.edu/dataset/165/concrete+compressive+strength

The dataset is embedded directly in `ml_pipeline/concrete_data.py` — no internet connection required to train the model.
